Privacy Policy for Read Later Is Broken

Last updated: June 6, 2026

Overview

Read Later Is Broken is a Chrome extension that helps you add the current browser page to selected NotebookLM notebooks. The extension is designed to use only the data needed for that user-facing purpose.

This project is not affiliated with Google or NotebookLM. NotebookLM is a service provided by Google, and your use of NotebookLM is governed by Google's applicable terms and privacy policies.

Data handled by the extension

The extension may handle the following data when you use it:

  • The URL and title of the currently active browser tab.
  • NotebookLM notebook names and notebook URLs retrieved from NotebookLM.
  • Your selected notebook destinations and digest notebook preferences.
  • The last add result, including status messages and the page URL/title involved.

The extension does not collect page body text, screenshots, passwords, payment information, precise location, contacts, or authentication credentials.

How data is used

The data is used to display destination choices in the popup, remember your selections, create or reuse digest notebooks, and send the current page URL to NotebookLM as a source when you choose to add it.

The extension does not use the data for advertising, analytics, profiling, or unrelated purposes.

Storage

Settings and the last add result are stored in Chrome extension storage on your browser profile. This includes saved NotebookLM destinations, selected destination IDs, digest checkbox settings, and the last add status.

You can remove this locally stored extension data by removing the extension or clearing extension storage through Chrome.

Data sharing

The extension does not send your data to a developer-operated server. When you use the extension to add a page, the selected page URL and related request data are sent to NotebookLM so NotebookLM can add the page as a source to your selected notebook.

The extension does not sell, rent, or transfer user data to advertising platforms, data brokers, or information resellers.

Chrome Web Store Limited Use statement

The use of information received from Google APIs or Google services will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

User data is used only to provide or improve the extension's single purpose: adding the current page to user-selected NotebookLM notebooks.

Remote code

The extension does not execute remotely hosted code. Extension code is bundled with the extension package. Network requests are made to NotebookLM only to perform user-requested actions.

Contact

For questions about this policy, use the support contact provided in the Chrome Web Store listing or the project contact channel linked from this site.
