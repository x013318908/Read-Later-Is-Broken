Permission Justifications

Last updated: June 6, 2026

Single purpose

Read Later Is Broken lets users add the current browser page URL to selected NotebookLM notebooks from a Chrome extension popup.

Requested permissions

activeTab

Used only after the user opens the extension popup to read the active tab's URL and title. This lets the extension add the current page to NotebookLM without requesting persistent access to all websites.

scripting

Used to run bundled extension helper code in temporary NotebookLM tabs. The helper code performs user-requested NotebookLM actions such as adding the current page as a source.

storage

Used to store extension settings in Chrome extension storage, including saved NotebookLM destinations, source counts, selected destination IDs, digest checkbox settings, and the last add result.

https://notebooklm.google.com/*

Required to interact with NotebookLM for the extension's single purpose: listing notebooks, showing source counts, creating notebooks, and adding the current page URL as a source to selected notebooks.

Single purpose description

Read Later Is Broken helps users add the current browser page URL to selected NotebookLM notebooks from a Chrome extension popup.

Remote code

No remote code is used. The extension executes only bundled extension code. Network requests are made to NotebookLM only to perform user-requested actions.

Data use summary

The extension handles the current tab URL/title, NotebookLM notebook names/URLs, source counts, saved destination selections, digest preferences, and the last add result. This data is used only to add the current page to user-selected NotebookLM notebooks, show destination status, and remember extension settings. The extension does not use this data for ads, analytics, profiling, or sale to third parties.

Not requested

The extension does not request broad <all_urls> host access, cookies, history, downloads, geolocation, microphone, camera, notifications, bookmarks, or declarative network request permissions.
