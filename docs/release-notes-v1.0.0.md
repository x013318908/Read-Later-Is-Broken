# Release Notes v1.0.0

## English

Version 1.0.0 is the first stable release of Read Later Is Broken.

What's new:
- Added source count badges for digest notebooks and saved theme notebooks.
- Improved notebook sorting: checked notebooks appear first, then notebooks are sorted by name.
- Kept emoji searchable while ignoring leading emoji for name sorting.
- Improved recovery when NotebookLM list, create, or add requests take too long.
- Disabled Enter submission in the shared search/new notebook name field.

## Japanese

バージョン 1.0.0 は Read Later Is Broken の最初の正式版です。

更新内容:
- ダイジェスト用ノートブックと保存済みテーマ別ノートブックに、ソース数バッジを追加しました。
- ノートブック一覧の並び順を改善し、チェック済みを上に、その中ではノートブック名順に並ぶようにしました。
- 先頭の絵文字は検索対象に残しつつ、名前順ソートでは無視するようにしました。
- NotebookLM の一覧取得、作成、追加リクエストが長時間返らない場合の復帰処理を改善しました。
- 検索欄兼新規ノートブック名入力欄で Enter を押しても送信されないようにしました。
